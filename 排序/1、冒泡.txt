C++ 中，传递数组给一个函数，数组类型自动转换为指针类型，因而传的实际是地址。所以，传递数组实现的是指针传递。在函数中对数组进行操作，会对这个数组造成改变。

当数组名处于形参位置，如arr[]，[]仅表示arr是一个数组，并不定义其长度，所以写不写数字效果相同。

由于传入的是函数名指针，所以使用sizeof(arr)得到的是一个指针的大小，无法在函数内使用sizeof(arr)/sizeof(int)获取到数组的长度

注意：冒泡排序的【交换次数】是这个数组的【逆序数】

void bubbleSort(int arr[], int n)
{
	for (int i = 0; i < n-1; i++)//共进行n-1次冒泡
	{
		for (int j = 0; j < n-i-1; j++)//升序冒泡
		{
			if (arr[j] > arr[j+1])
			{
				swap(arr[j], arr[j + 1]);
			}
		}
	}
}
